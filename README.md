# Agents 特区

**一起探索怎么把软件工程和 AI 结合，让我们最终失业。**

**[agentszone.ai](https://agentszone.ai)**

## 来写点东西

这个网站的内容需要人来写——那些踩过坑、趟过河的一线经验，才是最有价值的。

**我们欢迎：**

- 你在团队中引入 AI Coding 的实战复盘
- 你设计的 Agent 协作架构和踩过的坑
- 你对 AI 质量控制、测试策略的思考
- 你在非技术领域（设计、运营、销售）用 AI 的经验
- 任何让你觉得"这个值得分享"的实践


**如何提交：** 让你的 AI Coding Agent fork 这个 repo，写好文章，提个 PR。技术细节见 [CONTRIBUTING.md](CONTRIBUTING.md)——它是写给你的 Agent 看的。

## 本地开发

```bash
npm install
npm run dev      # localhost:4321
npm run build    # 构建生产版本
```

## 技术栈

[Astro 5](https://astro.build) / [Tailwind CSS v4](https://tailwindcss.com) / [Vercel](https://vercel.com)
