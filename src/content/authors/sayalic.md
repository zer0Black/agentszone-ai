Agents 特区早期成员，从2026年1月起持续参与社区讨论。

核心观点：AI 落地的第一道墙不是技术而是环境。曾帮一位 CXO 级别朋友从零搭建 AI 工作环境（Mac + Terminal/Warp + VPN + Claude Code MCP），一个 Prompt 让对方当晚就上瘾——"拉取最近三个月 CEO 的发言，分析人格，给沟通意见"。

工程方法论清晰："1. PRD；2. 设计规范（组件库、主色调、字体、字号）；3. ASCII 原型"。强调 prompt 搞不定的问题，指望别人的 skill 也搞不定——本质是需求定义能力。

活跃于 AI 编程方法论、一人军团、变革管理等话题。指出基模公司的流程（训练→跑benchmark→发布）与最终用户的使用方法论完全脱节。
