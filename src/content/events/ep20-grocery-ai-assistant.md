本期由米总（David）主讲，分享他在香港连锁生鲜门店落地 AI 管理系统的完整经验。

## 分享内容

- **问题背景** — 生鲜门店只有收银系统，没有库存管理、损耗追踪、进货计划。数据断点导致管理全靠感觉。
- **解决方案** — 用 AI 做"翻译官"：把店长的语音、图片翻译成后端 API 调用，通过 WhatsApp/Telegram 零门槛交互。
- **五个关键设计决策** — 使用门槛降到极致、Skill 固定功能范围、权限控制在 API 层面、架构分离、部署极简。
- **成本** — 云主机 + API 调用，月成本不到 300 元。

## 讨论亮点

- 这个模式适用于所有蓝领密集型企业的进销存场景
- 传统门店与 7-Eleven 的核心差距在信息采集能力
- 绿化采购询价领域已有类似实践验证
